# SBER File Manager

Python-утилита для управления файлами с автоматическим логированием перемещений через SQLite и Peewee ORM.

## Описание проекта

Этот инструмент предназначен для автоматического перемещения файлов между директориями с полным отслеживанием операций.

### Ключевые возможности:

✅ Автоматическое перемещение файлов  
✅ Отслеживание всех операций через базу данных  
✅ Детальное логирование событий  
✅ Обработка ошибок без потери данных  

## Технологии

| Компонент | Описание |
|-----------|----------|
| **Язык** | Python 3.x |
| **ORM** | Peewee |
| **База данных** | SQLite |
| **Шаблон** | shutil |
| **Логирование** | logging |

## Настройка

### Структура проекта

```
sber5/
├── README.md           # Эта документация
├── main.py            # Основной скрипт
├── files.db           # База данных
└── app.log           # Файл логов
```

Измените пути в `main.py`:

```python
first_path = "first"       # Откуда брать файлы
second_path = "second"     # Куда помещать файлы
```

## Использование

### Базовый запуск

```bash
python main.py
```

Скрипт выполнит:
1. Просканирует директорию first/ на наличие файлов
2. Переместит каждый файл во second/
3. Запишет информацию в базу данных files.db
4. Залоггирует операцию в app.log

## Расширенные настройки

### Фильтрация по расширению

```python
allowed_extensions = ['.pdf', '.docx', '.xlsx']
for i in listdir(first_path):
    filepath = joinpath(first_path, i)
    if isfile(filepath) and any(i.lower().endswith(ext) for ext in allowed_extensions):
        shutil.move(filepath, f'{second_path}/{i}')
```

### Частое выполнение (cron)

```bash
*/5 * * * * /usr/bin/python3 /path/to/main.py >> /var/log/sber5.log 2>&1
```

## Важные предупреждения

1. **Резервное копирование:** Перед массовым перемещением создайте резервную копию
2. **Проверка прав:** Убедитесь что у скрипта есть права на запись
3. **Конфликт имен:** Существующие файлы будут перезаписаны
